# html主体部分常用标签整理

控件标签

```
<select> <input type="date"/> <input type="time"/>
```

结构框架标签

```
<div> <class> 
```

引入标签

```
<p> <a> <h> <time> <img> <audio> <video>
```

功能标签

```
<br><hr> <strong> <ins> <em> <del>
```

列表标签

```
<ul> <ol> <li>
```

表格标签

```
<table> <caption> <th> <tr> <td> <fieldset> <legend>
```
